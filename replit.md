# VAD Operations Bot

The Replit Run workflow starts `cd bot && python main.py`.

The active bot provides role-aware creator, admin, and owner operations. Coffee Date
ordering is archived in `bot_backup_before_tracker/` and is never imported by the active
process.

Configuration belongs in Replit Secrets. Never store tokens or real owner IDs in files.
See `bot/README.md` and `docs/OPERATIONS_ROLLOUT.md` for environment keys, migrations,
rollback, permissions, and testing.
