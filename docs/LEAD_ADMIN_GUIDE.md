# Lead Admin Guide

Lead Admin tools cover day-to-day operational review: new creators, Away Requests, current
POP queue, creator reports, moderation, notes, and approved messaging. Available buttons are
filtered by assigned permission.

Lead Admins cannot access owner Setup, full audit identities, protected archives, exports,
system security details, or owner/access management. Escalate configuration, restoration,
and three-strike owner decisions to an Owner.
